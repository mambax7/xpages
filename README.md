# xpages
xPages - Advanced Static Page Management Module for XOOPS  xPages is a professional static page management module developed for XOOPS 2.7.0 and above. It makes content management easier with its user-friendly interface and powerful features.


🚀 Features
📄 Page Management

    Add, edit, delete pages

    Page status (Active/Inactive)

    Menu ordering and hierarchical page layout

    SEO-friendly URL (alias) support

    Short description and full content fields

    Page view counter (hits)

⚙️ Custom Fields

    10+ field types: Text, Textarea, Email, URL, Tel, Number, Checkbox, Radio, Select, File/Image

    Field scope: Global (all pages) or Page-specific

    Required field support

    Template display control

    File upload support (Image, PDF, DOC, ZIP)

🖼️ Image Gallery

    Add image gallery to pages

    Image title and description

    External URL or local upload support

    Drag-and-drop ordering

    Lightbox integration

🔍 SEO Support

    Meta Title, Meta Description, Meta Keywords

    Noindex / Nofollow controls

    SEO-friendly URL (alias)

    301 redirect support

    Automatic sitemap (sitemap.xml)

🎨 Template System

    Customizable Smarty templates

    Page list and single page templates

    Block support (Recent pages, Menu)

    Breadcrumb navigation

📊 Dashboard

    Statistics cards (Total pages, Active pages, Custom fields, Gallery images)

    Recent pages widget

    Most read pages widget

    Monthly page statistics chart

    Quick action buttons

🔧 Technical Features

    XOOPS 2.7.0+ compatible

    XMF Module Admin support

    Multi-language support (Turkish, English)

    Comment integration

    Search integration

    Block system
